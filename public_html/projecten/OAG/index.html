Projecten waar ik nu mee bezig ben;

Game voor Spannenburg.art

Opgeleverd:
In de zomer van 2020 ben ik begonnen met het leren van HTML. Al snel daar na ging ik samen met Arjan Spannenburg van start met de Online Adventure Game voor Spannenburg.art. Beginnend met het bedenken van een verhaal en gaande weg bedachten we puzzels en verdere inhoudelijke dingen. Dit resulteerde in een heleboel dingen in HTML continue aanpassen omdat ik ondertussen weer iets nieuws had geleerd.
Het idee van het spel is dat je tijdelijk Arjan de fotograaf komt vervangen. We zijn aan dit spel begonnen zodat ik een project had waar ik mijn nieuw geleerde HTML kon toepassen. Tegelijkertijd kan Arjan dit gebruiken als promotie voor Spannenburg.art.
