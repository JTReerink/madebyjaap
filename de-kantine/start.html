De Kantine

Waarom?

De verpleegsters hebben in hun kantine een zak met snoepjes verstopt. De code van het kastje staat op de tablet. Maar daar staat natuurlijk ook weer een code op. Los alle puzzeltjes en spelletjes op om zo in de snoepkast te komen.

Wat?

Op sommige schermen moet je een spelletje doen en sommige een puzzel. Als je het goed doet krijg je steeds een nieuwe code.

Hoe?

In de randen van de foto's zie je pijltjes staan. Klik daar op om zo door de kamer de kunnen bewegen. Wil je een iets van dichterbij bekijken? Klik dan op het voorwerp.
